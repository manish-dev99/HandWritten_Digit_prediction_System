# HandWritten_Digit_prediction_System

# 🤖 AI Digit Recognizer

An interactive **AI-powered web application** that recognizes handwritten digits using a **Convolutional Neural Network (CNN)** trained on the **MNIST dataset**.

Users can **draw a digit on a canvas or upload an image**, and the AI predicts the digit with confidence.

## 🧠 Model Details

The model is a **Convolutional Neural Network (CNN)** trained on the **MNIST Handwritten Digit Dataset**.

### Architecture

- Conv2D (32 filters, 3×3)
- MaxPooling2D
- Conv2D (64 filters, 3×3)
- MaxPooling2D
- Flatten Layer
- Dense Layer (64 neurons)
- Output Layer (10 classes for digits 0–9)
